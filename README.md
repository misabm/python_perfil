# Jogo Perfil

Este é um projeto desenvolvido em Python que recria o clássico jogo de tabuleiro **Perfil**. O objetivo do jogo é adivinhar o elemento secreto (Pessoa, Lugar, Coisa ou Ano) utilizando o menor número de dicas possível. O projeto foi construído focado em boas práticas de programação, tratamento de erros e Programação Orientada a Objetos (POO).

## 🚀 Funcionalidades Principais

*   **Modo Solo:** Jogue sozinho e avance pelo tabuleiro testando os seus conhecimentos.
*   **Modo Multiplayer:** Suporta de 2 a 6 jogadores reais no mesmo terminal, cada um com o seu próprio nick.
*   **Modo VS Computador (IA):** Enfrente a Inteligência Artificial com 3 níveis de dificuldade (Fácil, Normal e Difícil). A chance do robô acertar aumenta gradualmente a cada dica revelada!
*   **Tabuleiro Dinâmico:** Representação visual estilizada no terminal mostrando o progresso dos peões ao longo de 130 casas.
*   **Casas Especiais `?`:** 25 casas aleatórias do tabuleiro ativam um modo de jogo especial com regras restritas e pontuações diferenciadas.
*   **Sistema de Ranking:** Histórico persistente que armazena as pontuações e exibe o ranking final acumulado das partidas.
*   **Sistema Anti-Quebra:** Validação robusta de entradas de terminal que impede o fechamento do programa caso o utilizador digite caracteres inválidos.

## 💻 Como Executar o Projeto

### Pré-requisitos
Ter o **Python 3.x** instalado na sua máquina.

### Passo a Passo
1. Clone o repositório para a sua máquina local:
```bash
   git clone [https://github.com/misabm/python_perfil.git](https://github.com/misabm/python_perfil.git)
```
2. Acesse a pasta do projeto:
```bash
   cd python_perfil
```
3. Execute o arquivo principal:
```bash
   python main.py
```
